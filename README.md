# arcgis-vue-app

## Dependencies to be installed
esri-loader
```
npm install --save esri-loader
```

d3.js
```
npm install d3
```

jquery
```
npm install jquery
```

scss
```
npm install -D node-sass sass-loader
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
